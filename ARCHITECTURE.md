Implements Corrective RAG pattern using LangGraph for orchestration.

## Corrective RAG Flow

```
RU Query вҶ’ rewriteQuery (RUвҶ’EN) вҶ’ retrieve (BM25+vectorвҶ’RRF)
  вҶ’ gradeChunks (LLM: relevant? yes/no)
    в”ңв”Җ вүҘ2 relevant OR retriesвүҘ2 вҶ’ generate (answer in RU) вҶ’ return
    в””в”Җ <2 relevant AND retries<2 вҶ’ broadenQuery вҶ’ retrieve (loop)
```

## Hybrid Search

BM25 (keyword) + ChromaDB vector search вҶ’ Reciprocal Rank Fusion (k=60).
BM25 index is held in memory and rebuilt from ChromaDB on server startup.
